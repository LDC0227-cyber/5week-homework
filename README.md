# 글로벌 사전교육 5주차 과제

## 프로젝트 소개

이 프로젝트는 여러 AI Agent가 가상의 공간에서 생활하면서 서로 대화하고 관계를 형성하는 과정을 시뮬레이션하는 프로그램이고,

각 Agent는 이름, 나이, 직업, 성격 정보를 가지고 있으며, 대화를 통해 상대방에 대한 정보를 기억(Memory)하고 관계 정보(Reflection)를 업데이트한다.

또한 하루 일정을 스스로 계획하고 같은 장소에 있는 Agent들끼리 자동으로 대화를 진행한다.

---

## 개발 목적

대형 언어 모델(LLM)을 활용하여 사람 간의 상호작용을 모사하는 Generative Agent 개념을 이해하고 구현하는 것을 목표로 하였다.

특히 다음 기능에 중점을 두었다.

- Agent 프로필 관리
- Memory 저장 기능
- Reflection 기반 관계 관리
- 자동 일정 계획
- 장소 기반 상호작용
- 관계 리포트 생성

---

## 주요 기능

### 1. Agent 생성

각 Agent는 다음 정보를 가진다.

- 이름
- 나이
- 직업
- 성격

---

### 2. Memory 기능

대화 과정에서 얻은 정보를 저장한다.

예시

- 상대방의 직업
- 취미
- 관심사

---

### 3. Reflection 기능

상대방에 대한 관계 정보를 저장한다.

예시

- 친근한 사람
- 자주 만나는 사람
- 대화가 잘 통하는 사람

---

### 4. 자동 일정 생성

각 Agent는 하루 시작 전에 시간대별 활동 장소를 계획한다.

예시 장소

- 집
- 학교
- 카페
- 도서관
- 공원
- 레스토랑

---

### 5. 자동 대화

같은 시간대에 같은 장소에 있는 Agent들끼리 자동으로 대화를 진행한다.

처음 만나는 경우

- 인사
- 자기소개

기존에 알고 있는 경우

- 기존 Memory 기반 대화

---

### 6. 리포트 생성

시뮬레이션 종료 후 특정 Agent 시점에서 다음 정보를 출력한다.

- 알고 있는 사실(Fact)
- 관계 정보(Reflection)

리포트는 화면 출력과 파일 저장을 지원한다.
